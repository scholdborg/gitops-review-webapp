GitOps Review Webapp

A tiny static site that ships the GitOps way: every change is reviewed and built before it is allowed to go live.

The flow

A code change is automatically reviewed, then built, and only deployed to GitHub Pages if every check passes. Git is the source of truth, and GitHub Actions is the gate that decides whether the change reaches the live site.

Pipeline

Commit → Review → Build → Deploy

Status

Deployed from Git after checks pass
